Información de Formatos

DPT (Perkin Elmer)

Formato nativo de espectrómetros Perkin Elmer

índice wavenumber absorbance
1 3997.23779 0.99338

Soporta índice opcional en primera columna

CSV

Formato universal de valores separados por comas

wavenumber,absorbance
4000.0,0.5
3995.5,0.51

Con o sin encabezados

TXT

Texto plano con números separados por espacios

4000.0 0.5
3995.5 0.51
3991.0 0.52

Flexible y portable

{} JSON

Formato JavaScript Object Notation

{{ '{' }}wavenumbers: [...], absorbance: [...]{{ '}' }}

O array de objetos con keys

XLSX

Formato Excel (requiere npm install xlsx)

Columnas: wavenumber, absorbance

Instala librería SheetJS para soporte
